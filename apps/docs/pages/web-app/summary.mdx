# Web App

The Kafé web app is how users interact with the on-chain program.
It’s main purpose is to serve content to users. Using the web app, users will be able to perform 4 main actions:

- Browse and read guides
- Browse and vote on guide proposals
- Propose a guide
- Tip authors
